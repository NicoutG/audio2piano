# audio2piano
Convert any audio signal into an 88-key piano approximation using machine learning.

## Installation
Clone the repository and set up a virtual environment to run Audio2Piano.

    git clone https://github.com/ton-utilisateur/audio2piano.git
    cd audio2piano

### Create a Python virtual environment

#### Linux / macOS:

    python3 -m venv .venv
    source .venv/bin/activate

#### Windows (PowerShell):

    python -m venv .venv
    .venv\Scripts\Activate.ps1

### Install dependencies

    pip install --upgrade pip
    pip install -r requirements.txt
